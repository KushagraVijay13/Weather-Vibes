
# 🌤️ Weather-Vibes

Weather-Vibes is a clean, minimal, and responsive weather application that allows users to check real-time weather conditions for any city in the world.  
Powered by the OpenWeather API, it displays temperature, humidity, wind speed, and weather conditions — all wrapped in a simple and aesthetic UI.

---

## ✨ Features

- Search weather for **any city**
- **Real-time** weather data using OpenWeather API
- Shows:
  - Temperature 🌡️
  - Weather condition (Cloudy, Sunny, Rainy, etc.) ☁️
  - Humidity 💧
  - Wind speed 🍃
- Fully **responsive UI** (works on PC, tablet, and phone)
- Modern, clean design & smooth user experience 🎨

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/<your-username>/Weather-Vibes.git
````

### 2. Open the Project Folder

```bash
cd Weather-Vibes
```

### 3. Add Your API Key

Open `script.js` and replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

with your actual OpenWeather API key.

### 4. Run the App

Just open `index.html` in your browser.
No server required ✨
